# 🤖 Robot Command Simulator

This is a simple Python-based robot simulator that accepts user commands and updates the robot's position and direction.

## 🚀 Features
- Move Forward & Backward
- Turn Left & Right
- Tracks position (x, y)
- Displays direction (NORTH, SOUTH, EAST, WEST)

## 🧠 Concepts Used
- Python basics
- Functions
- Loops
- Conditional statements

## ▶️ How to Run
1. Run the Python file
2. Enter commands:
   - FORWARD
   - BACKWARD
   - LEFT
   - RIGHT
   - STOP

## 📌 Example Output
Position: (0, 1), Direction: NORTH
